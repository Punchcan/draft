##一.项目概述
###1.开发背景：
目标用户现有迫切的需要进行多店管理，并且本公司App产品整体的战略规划也需要满足BOSS、店长、造型师、收银员等不同角色的需求
###2.主要的产品功能：
此次的改版主要围绕着营帐统计与库存管理两个模块，设计了诸如销量统计、毛利统计、趋势对比、调货、进货方式设置等新功能点
###3.此版本的预期目标：
版本上线后可以使产品的功能更加全面，让用户更加有效的对公司进行信息化管理；为本公司扩展业务提供有力的支持
